* Builder Pattern

## 2. Structural Patterns
* Adapter Pattern
* Bridge Pattern
* Composite Pattern
* Decorator Pattern

* Strategy Pattern
* Template Method Pattern
* Visitor Pattern

# Setup Locally
```bash
git clone https://github.com/jojozhuang/design-patterns-java.git
mvn install
mvn test
```
Open the project in IntelliJ IDEA, compile and run the JUnit test cases.
